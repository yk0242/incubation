        ソーステキストの文法：
        -------------------
        普通の文字                   : そのまま表示されます。
        <<SOMETEXT>>             : SOMETEXTタイプの言葉・文節を入れるフィールドです。SOMETEXTが、上の入力エリアで使われる文字列です。
        <<SOMETEXT*LABEL>>  : 上に同じ。ただし、文字列を複数回、文中に使えます。
        <<*LABEL>>                  : 上の<<SOMETEXT*LABEL>>で入力した文字列を代入します。

        Notes: 
          -テキスト中に、『"』は直接書き込めません。&quot;　と書いて下さい。
          -テキスト中に、普通にHTMLタグを打っても、正しく機能します。（すると思いますが…^^；）ただし、タグのパラメータ内では、 " " ではなく ' ' を使って下さい。さらに、閉じタグは<\/TAG>のようにして下さい。
          -LABEL の形式は、t2、t17等のtと整数の形式、空の文字列、数字で始まる文字列にはしないで下さい。
          -LABEL およびHTMLタグとJS内の字のほとんどには、半角英数を使用して下さい。
          -<<SOMETEXT>>フィールド内の SOMETEXT は、異なるフィールド内で同じ文字列でも、別物として扱われます。
          -複数の<<SOMETEXT*LABEL>>フィールド内に異なる内容を入れるには、各々の LABEL の値は違っていなければいけません。
             例：もし <<SOMETEXT1*SOMELABEL>> が <<SOMETEXT2*SOMELABEL>>より前に現れたら、後者は <<*SOMELABEL>>として扱われます。
          -<<*SOMELABEL>>の前に、<<SOMETEXT*SOMELABEL>> が現れるべきです。もし<<*SOMELABEL>>が先に現れれば、それが<<SOMETEXT*LABEL>>系として扱われます。（SOMETEXTは空の文字列になります。）
          -作成ページ内の簡単なカスタマイズは、ｊｓスクリプトの最初の方にあるパラメータをいじれば出来ます。

               パラメータの翻訳：
　　　　　　　　var showScript = 0;           　　　  //1ならページ表示と同時に、表示エリアにソーステキストをそのまま表示します。
　　　　　　　　var rightMargin = 120;      　　　    //右マージンの幅（ピクセル数）；0だとマージンなし
　　　　　　　　var btn1val = "アドリブ！";       　 //アドリブ作成ボタンの表示
　　　　　　　　var btn2val = "表示クリア";  　　   //クリアボタンの表示
　　　　　　　　var bgcolor1 = "#C8D8EE";    　   //入力エリアの背景色
　　　　　　　　var bgcolor2 = "#AABBEE";    　   //出力エリアの背景色
　　　　　　　　var textColor1 = "#000000";     　 //入力エリアのテキスト色
　　　　　　　　var textColor2 = "#000000";     　 //出力エリアのテキスト色
　　　　　　　　var showTextColor = "#000077";   //ソーステキスト初期表示の、テキストの色（showScript==1の場合）
　　　　　　　　var showFldColor = "#FF0000";    //ソーステキスト初期表示の、フィールド部分の色（showScript==1の場合）

　　　　なお、日本語版の例を見るには、adlib_j.htmlをご覧下さい。（adlib_j.jsを使用）
